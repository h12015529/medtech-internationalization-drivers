# SME Internationalization

Author: Thomas Lackner

## Research Question
Welchen Einfluss haben Firmengröße und F&E-Intensität auf den Internationalisierungsgrad börsennotierter Medizintechnik-Unternehmen?

## Hypotheses
- H1: Der Internationalisierungsgrad (Auslandsumsatzanteil) steht in einem positiven Zusammenhang mit der Firmengröße.
- H2: Eine höhere F&E Intensität (F&E Ausgaben im Verhältnis zum Umsatz) geht mit einem höheren Internationalisierungsgrad einher.
- H3: Profitablere Med-Tech-Unternehmen weisen einen höheren Internationalisierungsgrad auf als wenigere profitable.
